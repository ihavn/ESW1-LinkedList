◆ list_ReturnCode_t

enum list_ReturnCode_t

Linked List return codes.

These are the codes that can be returned the linked lists functions.

Enumerator
LIST_OK 

Everything went well

LIST_EMPTY 

The List is empty

LIST_NOT_FOUND 

The item was not found in the list

LIST_NULL 

The list pointer is NULL

LIST_ERROR 

The function could not be completed due to memory limit
